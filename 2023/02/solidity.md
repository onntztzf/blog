# Solidity 从入门到放弃

## 是什么

`Solidity` 是一种面向对象的高级语言，它主要用于实现智能合约。

## 语言特性

1. 是一种针对以太坊虚拟机(EVM)设计的花括号语言
2. 是静态类型的，支持继承、库和复杂的用户定义类型

## 合约

一份合约就是以太应币应用的基本模块。

```solidity
pragma solidity ^0.4.19;

contract HelloWorld {

}
```

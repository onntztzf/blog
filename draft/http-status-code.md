# HTTP 状态码

## 1\*\*

代表请求已被接受，需要继续处理。

## 2\*\*

代表请求已成功被服务器接收、理解、并接受。

## 3\*\*

代表需要客户端采取进一步的操作才能完成请求。通常，这些状态码用来重定向，后续的请求地址。

## 4\*\*

代表客户端可能发生了错误，妨碍了服务器的处理。

## 5\*\*

代表服务器在处理请求的过程中有错误或者异常状态发生。
